# web2pdf

[![Build Status](https://travis-ci.org/Himenon/web2pdf.svg?branch=master)](https://travis-ci.org/Himenon/web2pdf)

Convert website to pdf (a4paper).

## Usage

```bash
web2pdf http://example.com output.pdf
```
